# TEMPLATE DES CAPACITÉS DES UNITÉS
# =================================
# 
# Ce fichier contient les 76 unités organisées par élément.
# Pour chaque unité, vous pouvez ajouter ses capacités de manière textuelle.
# 
# FORMAT SUGGÉRÉ POUR CHAQUE CAPACITÉ :
# - Nom de la capacité
# - Élément de la capacité
# - Type de cible (single_enemy, all_enemies, random_enemy, etc.)
# - Dégâts (ou 0 pour les capacités sans dégâts)
# - Effets (buffs, debuffs, effets spéciaux temporaires)
# - Cooldown
# - Description détaillée
#
# TYPES DE DÉGÂTS POSSIBLES :
# - "fixed": Dégâts fixes (ex: 50)
# - "attack": Dégâts basés sur l'attaque de la créature (ex: attack * 1.5)
# - "attack_plus": Attaque + bonus fixe (ex: attack + 30)
# - "defense": Dégâts basés sur la défense (ex: defense * 2)
# - "hp_percent": Pourcentage des PV max de la cible (ex: 15% des PV max)
# - "caster_hp_percent": Pourcentage des PV max du lanceur (ex: 10% des PV du lanceur)
# - "scaling": Dégâts qui évoluent avec le niveau/tour (ex: 20 + 5 par tour)
# - "random": Dégâts aléatoires (ex: entre 30 et 80)
# - "chain": Dégâts qui augmentent à chaque rebond (ex: 40 + 10 par rebond)
#
# TYPES DE CIBLES POSSIBLES :
# - "single_enemy": Un seul ennemi
# - "single_ally": Un seul allié
# - "self": Soi-même
# - "all_enemies": Tous les ennemis
# - "all_allies": Tous les alliés
# - "random_enemy": Ennemi aléatoire
# - "random_ally": Allié aléatoire
# - "chain_enemies": Chaîne entre ennemis (avec bounce_count)
# - "chain_allies": Chaîne entre alliés (avec bounce_count)
# - "front_row": Première ligne
# - "back_row": Ligne arrière
# - "adjacent_enemies": Ennemis adjacents
# - "adjacent_allies": Alliés adjacents
#
# EXEMPLES DÉTAILLÉS :
# Capacité 1: "Explosion de Feu"
#   - Élément: Feu
#   - Cible: all_enemies
#   - Dégâts: 80 fixed
#   - Effets: brûlure 15/tour pendant 3 tours
#   - Cooldown: 3 tours
#   - Description: Explosion qui touche tous les ennemis et applique un effet de brûlure
#
# Capacité 2: "Griffe Acérée"
#   - Élément: Neutre
#   - Cible: single_enemy
#   - Dégâts: attack + 25
#   - Effets: 
#   - Cooldown: 1 tour
#   - Description: Attaque physique renforcée avec les griffes
#
# Capacité 3: "Soin Vital"
#   - Élément: Eau
#   - Cible: single_ally
#   - Dégâts: 0
#   - Effets: soin 60 PV
#   - Cooldown: 2 tours
#   - Description: Soigne un allié de 60 PV
#
# Capacité 4: "Chaîne de Foudre"
#   - Élément: Foudre
#   - Cible: chain_enemies
#   - Dégâts: 50 + 15 par rebond
#   - Effets: 
#   - Cooldown: 4 tours
#   - Description: Éclair qui rebondit entre ennemis, augmentant à chaque rebond
#
# Capacité 5: "Armure Magique"
#   - Élément: Arcanique
#   - Cible: self
#   - Dégâts: 0
#   - Effets: +30 défense, +20% résistance aux sorts pour 3 tours
#   - Cooldown: 4 tours
#   - Description: Crée une armure magique qui protège contre les attaques
#
